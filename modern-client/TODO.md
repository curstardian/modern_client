🎮 Modern Client — 올인원 개요
🧭 프로젝트 정의

Modern Client는
👉 밝고 현대적인 UI를 가진 마인크래프트 런처 (클라이언트)다.

기존 런처들의 특징:

어둡고 복잡함
UI 구식 느낌
사용성 떨어짐

👉 Modern Client 목표:

“직관적이고, 빠르고, 보기 좋은 런처”

🎨 핵심 컨셉
🌈 디자인 방향
밝은 색 기반 (화이트 / 블루 / 소프트 컬러)
카드 UI (요소별 구분 명확)
큰 Play 버튼
미니멀 + 둥근 디자인
🧠 UX 목표
초보도 바로 사용 가능
클릭 2~3번으로 실행
설정은 깊게, 기본은 단순하게
⚙️ 기술 스택
💻 기본 구조
Electron → 앱 껍데기 (데스크탑 프로그램화)
HTML / CSS → UI
JavaScript (Node.js) → 기능

👉 이유:

프론트엔드 기반이라 개발 속도 빠름
UI 자유도 매우 높음
파일/프로세스 제어 가능


🎮 핵심 기능
1. 마인크래프트 실행
Java 실행 (child_process)
버전 선택 후 실행
JVM 옵션 설정 가능
2. 버전 관리
바닐라 / Forge / Fabric
자동 다운로드
버전 리스트 UI
3. 인스턴스 시스템
각각 다른 모드/설정
서버별 프로필 가능
4. 계정 시스템
오프라인 계정
(확장) Microsoft 로그인
5. 설정
RAM 설정
Java 경로
테마 (라이트/다크 가능)
🖥 UI 구조
🧭 기본 레이아웃
[ 좌측 메뉴 ] | [ 메인 화면 ]
6. 모드 관리
modrinth,curseforge api로 모드 원클릭 설치, 쉽게 관리, 
호환성 경고, 마인크래프트 실행할때 호환성 문제 있으면 모드 구동기가 먼저 경고하기 전에 깔끔한 ui로 아주 해결법 잘보이게 경고.
7. 모던 런처의 모드를 자동 실행되게 만들기

좌측:
- Home
- Instances
- Settings

메인:
- Play 버튼
- 선택된 인스턴스 정보
- 최근 실행 기록

🔥 차별화 포인트 (중요)

✨ “밝은 UI” (희귀함)
